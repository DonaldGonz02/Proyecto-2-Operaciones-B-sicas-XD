Operaciones Básicas

Proceso del Programa:

El software esta diseñado para capturar 2 valores enteros, realizar la operación que el Usuario indique y que muestre el resultado correspondiente a la operación seleccionada.

Los valores de entrada que requiere este programa son: 2 variables(a y b) con valores enteros.

Capturar los valores ingresados para a y b y operarlos de acuerdo con lo que indique el Usuario, y luego enviarlo a la variable establecida para el resultado.
